# n8n Workflow Yedekleri

Bu klasör, projeye ait tüm n8n akışlarının (workflow) JSON yedeklerini içerir.

## Nasıl Dışa Aktarılır?

1. n8n arayüzünü tarayıcıda açın.
2. Kaydetmek istediğiniz akışa girin.
3. Sağ üst köşedeki menüden (⋮) **"Download"** seçeneğine tıklayın.
4. İndirilen `.json` dosyasını bu klasöre kopyalayın.

## Nasıl İçe Aktarılır?

1. n8n arayüzünde **"Import from file"** seçeneğini kullanın.
2. Bu klasördeki ilgili `.json` dosyasını seçin.
